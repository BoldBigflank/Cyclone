Amazon Head Tracking Unity Plugin Version 1
Amazon Motion Gestures Unity Plugin Version 1

Release Notes:
Revision 1 - June 18, 2014

This plugin exposes the HeadTracking and MotionGesture APIs for use in Unity.
Sample applications in the form of scripts are included to demonstrate basic usage of those APIs.
